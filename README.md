> # Para excutar

```bash
# Clone Repository
$ git clone https://github.com/epapini/IGNITE-dtMoney.git
```

```bash
# Install Dependencies
$ yarn
```

```bash
# Run Aplication
$ yarn start
```

```bash
# Result
http://localhost:3000/
```
